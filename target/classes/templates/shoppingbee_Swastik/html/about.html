Shoppingbee

Shoppingbee is developed by Swastik Roy at Beehyv Software Services as a part of training assignment. It is designed in such a way that it can be used across any platform maintaining its responsive design.

Features
  • 1. Scroll to reduce the height of navbar like major ecommerce websites
  • 2. Get dynamic url in catalog page when any filter is selected.
  • 3. On refresh or on using back button to catalog page the filters will be retained
  • 4. Scroll position is retained on refresh
  • 5. Add or remove items from cart
  • 6. The catalog page is fully responsive
  • 7. This is a static website and all data are retrieved from static json objects or window.localstorage
  • 8. Tags are added dynamically when any filter is clicked
  • 9. Search items using the search function in navbar
